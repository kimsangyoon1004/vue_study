# directive - Vue 디렉티브 연습

> `v-` 접두사로 시작하는 Vue 디렉티브들을 하나씩 연습합니다.

---

## 디렉티브란?

- DOM을 **조작**하거나 **반응형**으로 동작하게 하는 특수 속성
- `v-` 접두사가 붙음

---

## 학습 목록 (차례대로 추가)

### 조건부 렌더링

| 디렉티브 | 설명 | 학습 |
|----------|------|------|
| `v-if` | 조건이 false면 DOM에서 제거 | ⬜ |
| `v-else-if` | v-if 체인 | ⬜ |
| `v-else` | v-if의 그 외 경우 | ⬜ |
| `v-show` | 조건이 false면 `display: none` (DOM은 유지) | ⬜ |

### 리스트 렌더링

| 디렉티브 | 설명 | 학습 |
|----------|------|------|
| `v-for` | 배열/객체 반복 | ⬜ |

### 이벤트 / 속성 바인딩

| 디렉티브 | 설명 | 학습 |
|----------|------|------|
| `v-on` | 이벤트 리스너 | ⬜ |
| `v-bind` | 속성 바인딩 | ⬜ |
| `v-model` | 양방향 바인딩 | ⬜ |

---

## v-if vs v-show 차이

| 구분 | v-if | v-show |
|------|------|--------|
| false일 때 | DOM에서 **제거** | `display: none` (DOM 유지) |
| 토글 빈도 | 자주 바뀌면 비효율 | 자주 바뀌어도 OK |
| 초기 렌더 비용 | 낮음 (안 보이면 안 그림) | 높음 (항상 렌더) |
