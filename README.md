# vue_practice - Vue 3 학습 정리

---

## 폴더별 학습 목차

| 폴더 | 다루는 내용 |
|------|--------------|
| **vue란** | Vue 기본 개념 (프레임워크 소개, Options API vs Composition API, SFC 등) |
| **directive** | 디렉티브: v-if, v-show, v-for, v-on, v-bind, v-model, Mustache `{{ }}` |
| **methods** | Options API 의 `methods` – 함수 정의 및 template 에서 호출 |
| **ref** | Composition API: ref, reactive, computed 기본 |
| **computed** | computed – 계산된 값, 캐싱 |
| **watch** | watch – 값 변경 감시 (newValue, oldValue) |
| **watch_immediate** | watch 옵션: immediate, deep |
| **modifier** | 이벤트 수정자: .once, .prevent, 키 수정자 (.enter, .s 등) |
| **dynamic_ref** | v-bind (:class) 로 class 동적 바인딩 |
| **defineProps** | 부모 → 자식 props 전달 (defineProps) |
| **defineEmits** | 자식 → 부모 이벤트 (defineEmits, emit) |
